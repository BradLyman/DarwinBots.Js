(require("Parser/SourceManager"))(rawSource) → {SourceManager}

Creates a new SourceManager instance. The SourceManager helps with parsing source code by keeping track of how far through the code we have parsed (using the cursor) and what line we are currently on. Helper methods are provided for skipping whitespace, and matching keywords.

ExpectCallback(key, errName) → {Result}

Matches a regex against the next non-whitespace characters in the source code. If a match is found then the matched string is returned and the cursor is incremented, otherwise an error is returned and the cursor remains untouched.
